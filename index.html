Hangman

The user will have be able to guess the word on the screen letter by letter. If the user chooses a letter that is not in the word or a letter that has been guess already they lose a try. If they choose a letter that is in the word the letter will appear in the corresponding place of where it is in the word. The user must guess all the letters in the word in order to get a point added to their wins. If the user fails to guess the word in the amount of tries given then another word will populate and there wins will not increase nor decrease.
